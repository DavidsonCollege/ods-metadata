Reporting View Target: COURSE_CATALOG

Description Contains details for a subject (course) offered as part of the instruction in an academic study program. Details tracked will be specific to each academic period for which the subject could be offered. Academic periods will end when the course has an end academic period or when the status associated with the subject makes it not available to be scheduled.
Key & Frequency One row per subject and course number per academic period in which the course was active.
Recommended Search Columns SUBJECT, COURSE_NUMBER
