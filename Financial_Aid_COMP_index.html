Financial Aid Composite View Meta Data Reports

Target Reports:

Composite ViewDescription
AR_APPLICANT_NEED
Source composite view contains financial aid applicant need information for an aid year.
AR_APPLICANT_RESOURCE
Contains Financial Aid resource information and amounts available to financial aid applicants.One row per aid year per person per resource type per resource number.
AR_AWARD_BY_AID_YEAR
Contains detail data for each person who was made an offer of financial aid within the aid year.One row per person per aid year per fund.
AR_AWARD_BY_FUND
Source composite view contains each fund with its award information for each time indication.
AR_AWARD_BY_PERSON
Source composite view contains each applicant and award information for each fund and time indication.
AR_AWARD_DISBURSEMENT
Source composite view contains each disbursement for each applicant award for each fund for each academic period.
AR_FINAID_APPLICANT_STATUS
Source composite view contains each financial aid applicant with the latest application information for each aid year identified by the persons who have a financial aid status or RORSTAT record for the aid year.
AR_FINAID_APPLICATION1
Contains information pertaining to financial aid needs analysis.One row per Aid Year per Person per Interface Tape Code per Application Number.
AR_FINAID_APPLICATION2
Contains demographic information about the student and parents from the financial aid application. Also contains information specific to the financial aid application itself.
AR_FINAID_APPLICATION3
Contains information specific to the federal methodology for computing applicant need.One row per Aid Year per Person per Interface Tape Code per Application Number.
AR_FINAID_APPLICATION4
Contains information specific to the institutional methodology of computing applicant need.One row per Aid Year per Person per Interface Tape Code per Application Number.
AR_FINAID_BUDGET_COMPONENT
Source composite view contains each applicant with the budget information broken down into each component.
AR_FINAID_COMMENTS
Contains comments associated with a financial aid applicant.One row per person per sequence number.
AR_FINAID_ENROLLMENT
Source composite view contains financial aid recorded student enrollment summary information by aid year, academic period and enrolment rule.
AR_FINAID_FUND
Source composite view contains each fund with its detailed information for each aid year.
AR_FINAID_HOLD
Contains all of the financial aid holds (restrictions) for a person.One row per person per hold per hold from date per hold to date.
AR_FINAID_MESSAGE
Contains message information provided to financial aid applicants.One row per Aid Year per person per message code
AR_FINAID_TRACKING_REQUIREMENT
Source composite view contains tracking requirements for financial aid per person.
AR_FUND_MESSAGE_RULE
Contains information that displays the links between financial aid messages and specific funds and then displays the related messages provided to financial aid applicants..One row per aid year per fund per message per pidm
AR_LOAN_AGGREGATES
Contains information received from the National Student Loan Data System (NSLDS) such as NSLDS summary and name information, and the Stafford, Direct, and Perkins loan aggregate and history information.
AR_LOAN_APPLICATION
Source composite view contains loan application information for each person, fund and application number.
AR_LOAN_DISBURSEMENT
The Loan Disbursement by Academic Period contains detail data for loan disbursements that were made within the term. Details may include person and person address; loan fund code and description; loan period and description; loan application number; term recommended, expected and approved loan amount; loan status code; term disbursement date, sequence number, reference number, check number and amount; feed indicator; disbursement load; loan fee amounts; and disbursement rejects and edit fields.One row per person per academic period per fund per application number per disbursement date per disbursement.
AR_NEED_ANALYSIS_ISIR_SPEC
Need analysis data specific to the Institutional Student Information Report.One row per Aid Year per Person per Interface Tape Code per Application Number
AR_NEED_ANALYSIS_OVERRIDE
This view returns rows of override information for aid year and person.One row per aid year per person.
AR_NEED_ANALYSIS_PROFILE_SPEC
Additional Institutional Methodology Data.One row per Aid Year per Person per Interface Tape Code per Application Number.
AR_SATISFACTORY_ACAD_PROGRESS
Source composite view contains satisfactory academic progress information recorded within financial aid for the student.
AR_STUDENT_PERIOD_DEFINITION
Contains crosswalk data for each student that has a record in Financial Aid. Purpose of the crosswalk is to be able to determine an Aid Year or Period of Enrollment for a student's academic period.
AR_USER_DEFINED_FIELDS
Source composite view contains user defined fields for each aid year and person.
AR_VALIDATION
Source composite view contains a number of the source systems' validation table records. This view was meant to be used with reporting tools to create list of values as well as for cleansing purposes in the Enterprise Data Warehouse.
