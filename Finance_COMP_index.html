AF_ACCOUNT_ATTRIBUTES_ODS
Source composite view contains all the attributes (sets, types, values) associated with an account code.
AF_ACCOUNT_HIERARCHY
Source composite view contains the account hierarchy.
AF_ACCOUNT_HIERARCHY_FISCAL
Source composite view contains the account hierarchy by fiscal year and fiscal period. The composite view build is controlled by a parameter setting with an Internal Group = 'ODS EXTRACT PARAMETERS', Internal Code 1 = 'HIERARCHY_FISCAL'. The External Code should be set to the first fiscal year that was posted in the system or to the first fiscal year that you want the original hierarchy to display. The Description value will control how far out to build the view for a fiscal year/period as of today. If the value is set to 2, the system builds the fiscal year/periods 2 years past today's date.
