Composite ViewDescription
AN_EMPLOYEE_POSITION
Source composite view contains data on each position or assignment held by the employee.
AN_FISCAL_YEAR
Source composite view contains all fiscal years that have current data.
AN_FMLA_LEAVE_COMMENT
Contains text lines associated with the family medical leave base record.One row per person per family medical leave per comment
AN_JOB_FTE_TRACKING
Source composite view contains job related full-time equivalency (FTE) information for an employee within a position.
AN_LABOR_COST_DISTRIBUTION
Source composite view contains the accounting details for the labor distribution associated with an employee in a position or assignment.
AN_PAYROLL_DISTRIBUTION
Contains organization payroll distribution information specific to each payroll.
AN_PERSONNEL_ACTION

AN_PERSONNEL_ACTION_COMMENT
Contains text lines associated with the personnel action (PAF) transaction number for the person.One row per person per PAF transaction number per PAF sequence number per comment
AN_PERSONNEL_ACTION_EARNING
Source composite view contains default earning code changes associated with personnel actions (PAF) transaction number for the employee over their time. Includes personnel actions with current status for applied and pending transactions.One row per person per PAF transaction number per approval type per position per suffix per earning code per effective date.
AN_PERSONNEL_ACTION_GENERAL
Source composite view contains data on each personnel action (PAF) transaction number for the employee, position and job suffix where there are updates to biographical, employee or job information..One row per person per PAF transaction number per approval type per position per job suffix per apply column
AN_PERSONNEL_ACTION_LABOR_DIST
Source composite view contains labor distribution changes associated with personnel actions (PAF) transaction number for the employee over their time. Includes personnel actions with current status for applied and pending transactions.One row per person per PAF transaction number per approval type per position per suffix

Source composite view contains budget funding information for specific positions.
AN_POSITION_DEFINITION
Source composite view contains details associated with a position whether posted or filled.
AN_POSITION_LABOR_DISTRIBUTION
Source composite view contains the accounting details for the labor distribution associated with a position.
AN_SALARY_RATE
Source composite view contains salary table information for specified salary groups.
AN_SUPERVISOR
Provides information about employees' supervisors by first selecting the specific job data from the Assignment Repeating Table but when it is blank using a function to select the person who has an assignment that equals the position noted as the supervisor for the Postion Definition or base table. Note if supervisor job assignments are being maintained on the Assignment Repeating Table, they will be available and display even when the supervisor on the assignment record has been terminated. Data derived using the Position Definition will exclude terminated job assignments when displaying a current supervisor on the Employee reporting view. Employee Postion will contain all assignment records with their status including those where either the employee and or the supervisor are terminated.
AN_VALIDATION
Source composite view contains a number of the source systems' validation table records. This view was meant to be used with reporting tools to create list of values as well as for cleansing purposes in the Enterprise Data Warehouse.
AP_BARGAINING_UNIT
Source composite view contains bargaining unit information relating to an employee.
AP_BENEFICIARY_DEPENDENT
Source composite view contains beneficiary and benefit information for an employee.
AP_BENEFIT_DEDUCTION
Source composite view contains benefit and deduction information for an employee. Data includes information on benefits and deductions, coverage options and begin and end dates. The view returns one row per person per benefit per effective date.
AP_CERTIFICATION
Source composite view contains certification information relating to an employee.
AP_EARNING_TOTALS_CY
Source composite view contains earnings total information relating to an employee for a calendar year.
AP_EARNING_TOTALS_FY
Source composite view contains earnings total information relating to an employee for a fiscal year.
AP_EMPLOYEE
Source composite view contains basic information pertaining to employees. Data includes status, employee class, home organization, leave and benefits categories, etc. The view returns one row per person displaying the primary position or assignment.
AP_EMPLOYEE_EARNING_CY
Source composite view contains pay earnings and hours information for an employee by calendar year.
AP_EMPLOYEE_EARNING_FY
Source composite view contains pay earnings and hours information for an employee by fiscal year.
AP_FACULTY_APPOINTMENT_HISTORY
Provides historical information about faculty appointments.
AP_FACULTY_RANK_HISTORY
Provides historical information about faculty rankings.
AP_FACULTY_SABBATICAL_HISTORY
Provides historical information about faculty sabbaticals.One row per person per action date.
AP_FAMILY_MEDICAL_LEAVE_HIST
Source composite view contains details for the employees who are on family medical leave.one per person per family medical leave per earnings
AP_HR_APPLICATION
Source composite view contains application information for persons that are applying for a position at the institution. Data includes information pertaining to requisition number, position, class, application date, skill, source, etc. The view returns one row per position applied for.
AP_HR_APPLICATION_STATUS
Source composite view contains status information for applicants.
AP_HR_REQUISITION
Source composite view contains information pertaining to an HR Requisition.
AP_INTERVIEW
Source composite view contains application interview information for persons that are applying for a position at the institution.
AP_LEAVE_ACCRUAL
Contains leave accrual information specific to each payroll.
AP_LEAVE_BALANCE
Source composite view contains leave balance information relating to an employee.
AP_MONTHLY_DEDUCTION
Source composite view contains deduction information relating to an employee.
AP_PAYROLL_ATTENDANCE
Contains attendance information pertaining to specic pay periods.One row per calendar year per payroll identifier per payroll number per person per position per suffix per effective date per earnings per attendance date
AP_PAYROLL_DEDUCTION
Contains payroll deduction information specific to each payroll.
AP_PAYROLL_DOCUMENT
Contains payroll document (check and direct deposit) information specific to each payroll.
AP_PAYROLL_EARNING
Contains earnings information specific to each payroll.
AP_PAYROLL_EMPLOYEE_POSITION
Contains payroll history and related job history information specific to each payroll.
AP_PAYROLL_LABOR_DIST_OVERRIDE
Contains labor distribution override information pertaining to specic pay periods.One row per calendar year per payroll identifier per payroll number per system generated indicator per person per event sequence number per position per job suffix per position begin date per earnings per shift number.
AP_PAYROLL_TIMESHEET
Contains timesheet information pertaining to specic pay periods.One row per calendar year per payroll identifier per payroll number per system generated indicator per person per event sequence number per position per job suffix per position begin date per earnings per shift number per time entry period begin date.
AP_REFERENCE
Source composite view contains references information for an employee or person applying for a position at the institution.
AP_REVIEW
Source composite view contains information pertaining to employee performance reviews.
AP_TAX
Source composite view contains tax deduction information for an employee. Data includes information pertaining to the tax deduction, deduction amounts, Canadian deduction amounts, filing status, exemptions and deduction options.
AP_VALIDATION
Source composite view contains a number of the source systems' validation table records. This view was meant to be used with reporting tools to create list of values as well as for cleansing purposes in the Enterprise Data Warehouse.
AP_YEARLY_DEDUCTION
Contains summarized payroll deduction information for each quarter and year-to-date (YTD).
